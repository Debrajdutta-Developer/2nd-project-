# Sales Performance Analysis

## 📊 Objective
Analyze sales data to identify top-performing regions, products, and trends.

## 🧹 Data Cleaning
- Removed duplicates
- Handled missing values
- Formatted date columns

## 📈 KPIs
- Total Revenue
- Total Profit
- Average Order Value

## 📊 Visualizations
- Sales by Region (Bar Chart)
- Sales Over Time (Line Chart)

## 🔍 Insights
- West region has highest sales
- Laptop category generates most revenue
- Some regions show low performance

## 💡 Recommendations
- Focus marketing on top regions
- Improve low-performing areas
- Promote high-demand products
- Optimize pricing strategy
- Use seasonal campaigns
